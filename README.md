### Build
Dependencies : 
- libsdl2-dev
- libsdl2-image-dev
- libsdl2-ttf-dev

```bash
$ sudo apt install libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev
```

To build  : `make`

### Documentation

Documentation is here : [bomberman.matteyeux.com](http://bomberman.matteyeux.com)

You can generate doc with : `make docs`.
